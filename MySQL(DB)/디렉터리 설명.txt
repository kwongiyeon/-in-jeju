데이터베이스
8주차 과제
Community 프로젝트에서 사용한 Express와 MySQL을 연결하고, json 대신 sql 데이터를 가져오도록 만듭니다.
프론트엔드와 연결하지 않고 express -> Spring으로 마이그레이션합니다.

Express = Node.js 웹 애플리케이션 프레임워크(백엔드 서버)
MySQL = 오픈 소스 관계형 데이터베이스 관리 시스템(RDBMS)

expressjs 에 있는 라우터를 스프링 부트에서 하나씩 재구현 하는 작업입니다.
expressjs 쪽에 있는 router 를 스프링쪽에 구현하시고 expressjs 라우터 마다 있는 개개별 로직은 스프링에서 똑같이 구현하시면 됩니다!
다만 디비 관련 jdbcTemaplate 을 사용해서 기능을 구현해주시고 쿼리가 익숙해지시고 나면 JPA, QueryDSL 을 하나씩 적용해나가시는게 좋습니다.
라우터 한방에 다 옮기고 로직 다옮기고 하는것 말구 라우터 하나를 잡고 그 라우터에 딸려 있는 모든 기능을 하나씩 스프링에서 재구현하시는 과정입니다~
=> 백엔드 서버만 스프링으로 사용
